Math Ninja

Math Ninja is a concept design for an app intended to help students refresh their math skills by focusing on ways to improve memory by breaking down information so it is easily recalled. In addition it would take into consideration the idea that all people learn in a variety of ways, and looking at problems in multiple ways can enhance memory. With this in mind, this application combines the ideas of experiential learning, reading, speaking, and listening.

Looking back on this project I would add the ability for users to connect with each other and accumulate points. These features would work to create an app community where users are motivated by their friends to complete tasks.

Escape Room OS X App

AI is an OS X application that was used while running the Cal Poly Escape Room during Fall 2015. Participants would enter a room with a variety of puzzles hidden under desks, in locked cabinets, and behind posters. Once the puzzle answers were found and completed the participants would use AI to enter their answers. Each puzzle corresponded to an icon displayed on AIâ€™s screen. If the answer was correct, the icon puzzle would light up and the participants would know they were one step closer to escaping.

AI was written using Swift2 and Xcode. The original app was created by Jacob Johannessen. Jason Lu and myself were tasked with finding ways to improve its functionality. We decided to implement an admin panel so non technical users would be able to create new puzzles and change icon images without having to change code.

  • Class: Liberal Arts And Engineering Studies 301
  • Date: Fall 2015
  • Collaborators: Jacob Johannesen, Jason Lu
